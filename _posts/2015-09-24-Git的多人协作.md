为了写实验室的unity项目，与同学试了一下git的协作。对方并不太熟悉git，branch部分的暂时先可以不用，陆续更新，以后再补上。     

```sh     
git pull origin  //别人push过以后的更新     
git log  //记录下最新的commit序列号     
git show serial_of_commit  //可以看到修改的内容     
```
